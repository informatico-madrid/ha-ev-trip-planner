# Engram SDK - Memoria Persistente

## Instalación
- Node.js v22.22.2 (instalado via fnm)
- engram-sdk v0.6.1 (global, npm)

## Configuración
- API Key Gemini: Configurada y guardada
- Vault: ~/.engram/default.db
- Integración: Claude Code + VS Code

## Herramientas Disponibles
1. engram_remember - Guardar memorias
2. engram_recall - Buscar memorias relevantes
3. engram_surface - Contexto proactivo
4. engram_briefing - Briefing de inicio
5. engram_consolidate - Consolidación final
6. engram_connect - Conectar memorias
7. engram_forget - Eliminar memorias
8. engram_entities - Listar entidades
9. engram_stats - Estadísticas vault
10. engram_ingest - Extraer de texto

## Notas Importantes
- Reiniciar Claude Code/VS Code para activar
- Las memorias de esta sesión no persisten hasta reinicio
- Auto-consolidación habilitada al finalizar sesión
