## Learnings

- **Discovery**: EV Trip Planner panel usa Lit v2.8.0 con Shadow DOM OPEN por defecto
- **Confir**: Selector `>>` de Playwright SÍ funciona para atravesar Shadow DOM de Lit
- **Code**: Panel está en `/home/malka/ha-ev-trip-planner/custom_components/ev_trip_planner/frontend/panel.js`
- **Pattern**: Tests actuales (trip-crud.spec.ts, trip-states.spec.ts) usan correctamente `ev-trip-planner-panel >> .add-trip-btn`
- **Risk**: Si futuro cambio a `shadowOptions: { mode: 'closed'}`, `>>` dejaría de funcionar
- **Recommend**: Usar `data-testid` en componentes Lit para selectors más estables
