## Acceptance Criteria

1. AC-1: Automation reads p_deferrable{n} from EMHASS plan for each hour
2. AC-2: If p_deferrable{n} > 100W and car home+plugged → start charging
3. AC-3: If p_deferrable{n} == 0W and charging active → stop charging
4. AC-4: If car not home or not plugged when EMHASS schedules → notify user
5. AC-5: Manual mode activated → automation doesn't interfere
