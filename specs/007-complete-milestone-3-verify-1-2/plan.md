    "charging_power_kw": float,
    "profile": List[float]  # Power in Watts for each hour

### Test Contracts

#### Contract: Trip Calculation Functions

**Functions to Test**:
- `async_get_next_trip()` - Returns next trip or None
- `async_get_kwh_needed_today()` - Returns total kWh for today's trips
- `async_get_hours_needed_today(charging_power_kw)` - Returns hours needed to charge

**Test Coverage**:
- Empty trip list → None / 0 / 0
- Single trip → correct values
- Multiple trips → sum of values
- Mixed recurring/punctual → correct next trip

#### Contract: SOC-Aware Power Profile

**Functions to Test**:
- `async_generate_power_profile()` with vehicle_config
- SOC above threshold → no charging scheduled
- SOC below threshold → charging scheduled to reach minimum

**Test Coverage**:
- SOC = 80%, needs 7.5 kWh → no charging (24.5 kWh remaining > 16 kWh minimum)
- SOC = 49%, needs 7.5 kWh → charging scheduled (12.1 kWh remaining < 16 kWh minimum)
