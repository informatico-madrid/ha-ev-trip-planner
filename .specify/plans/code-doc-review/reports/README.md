# Documentation Audit Reports

This directory contains reports generated by the documentation audit process.

- `audit-report.json`: Machine-readable report of all path references found in documentation.
- `audit-summary.md`: Human-readable summary of audit findings and recommended fixes.

All reports are generated automatically by the audit script and should not be edited manually.
